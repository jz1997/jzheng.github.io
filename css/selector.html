基本选择器

选择器 示例 描述
* * 选择所有元素
#id #title 选择 id="title" 的所有元素
.class .introduction 所有 class="introduction" 的元素
element p 选择所有 p 元素
element,element div, p 选择所有 div 和 p 元素
element element div p 选择 div 子元素（包含孙子元素）中的所有 p 元素
element>element div>p 选择父元素为 div 元素的所有 p 元素 （子元素）
element+element div+p 选择紧接在 div 元素之后的 p 元素
element~element p~ul 选择和 p 元素同级并且在 p 元素之后的 ul 元素

属性选择器

选择器 示例 描述
[attribute] a[target] 带有 target 属性的所有 a 元素
[attribute=value] a[target="_blank"] target 属性等于 "_blank" 的所有 a 元素
[attribute~=value] a[class~="hello"] class 属性包含 "hello" 单词的所有 a 元素，例如： 
<a class="hello world">点击跳转</a>
[attribute|=value] a[target|="hello"] target 属性为 hello 或者 hello-* 所有 a 元素
[attribute*=value] a[href*="example"] href 属性值包含 "example" 字符串的所有 a 元素, 例如： 
<a href="https://www.exmaple.com">点击跳转</a>
[attribute^=value] a[href*="https"] href 属性值以 "https" 字符串的开头的所有 a 元素, 例如： 
<a href="https://www.exmaple.com">点击跳转</a>
[attribute$=value] a[href$=".com"] href 属性值以 ".com" 字符串的结尾的所有 a 元素, 例如： 
<a href="https://www.exmaple.com">点击跳转</a>

伪类选择器

状态 示例 说明
:link a:link 选择未被访问的链接
:visited a:visited 选择已被访问的链接
:hover a:hover 鼠标移动到元素上时
:active a:active 点击正在发生时
:focus input:focus 选择正在获取焦点的元素
:root :root 选择文档的根元素，即 html 元素，可以用于设置全局 css 变量，或者应用全局样式
:empty div:empty 用于选择没有子元素的元素
:target div:target 用于访问锚点目标元素(要访问锚点才会显示)
:first-child p:first-child 选择所有父元素的第一个子元素的所有的p元素
:last-child p:last-child 选择所有父元素的最后一个子元素的所有的p元素
:first-of-type p:first-of-type 选择所有父元素的首个p元素的每个p元素
:last-of-type p:last-of-type 选择所有父元素的最后p元素的每个p元素
:only-of-type p:only-of-type 选择所有父元素的唯一的p元素的每个p元素
:only-child p:only-child 选择所有父元素的唯一元素的每个p元素
:nth-child(n) p:nth-child(2) 选择所有父元素的第二个子元素的每个p元素
:nth-last-child(n) p:nth-last-child(2) 同 nth-child(n) 从最后一个开始计数
:nth-child(odd) p:nth-child(odd) 选择所有父元素的奇数p元素
:nth-child(even) p:nth-child(even) 选择所有父元素的偶数p元素
:nth-of-type(n) p:nth-of-type(2) 选择所有父元素的第二个p元素的每个p元素
:nth-last-of-type(n) p:nth-last-of-type(2) 同 nth-of-type(n) 从最后一个开始计数
:not(selector) :not(p) 选择非p元素的所有元素

表单伪类选择器

状态 示例 说明
:enabled input:enabled 选择每个启用的 input 元素
:disabled input:disabled 选择每个禁用的 input 元素
:checked input:checked 选择每个被选中的 input 元素
:required input:required 包含 required 属性的 input 元素
:optional input:optional 不包含 required 属性的 input 元素
:valid input:valid 验证通过的表单元素
:invalid input:invalid 验证不通过的表单元素
